Halaman ini berisi beberapa project Internet of Things (IoT) yang pernah saya buat menggunakan ESP32, Arduino, dan sensor.

Jammer wifi & bluetooth(ESP32)

Jammer WiFi dan Bluetooth (sering disebut WiFi jammer dan Bluetooth jammer) adalah alat yang mengganggu sinyal agar perangkat tidak bisa terhubung ke jaringan.

Monitoring Pergerakan Di area sekitar

Alat ini dapat mendeteksipergerakan manusia dan lainnya yang dapat di ketahui dengan buzzer yang berbunya dan mengirimkan data di telegram anda.

Kontrol LED via Web

LED dikontrol melalui browser menggunakan ESP32 sebagai web server tanpa aplikasi tambahan.

Alarm Sensor Gerak

Menggunakan sensor PIR untuk mendeteksi gerakan. Jika ada gerakan, buzzer menyala dan notifikasi muncul.
